# Quick Installation Guide for Translation Feature

## Step 1: Install Backend Packages

Run this command in the backend directory:

```bash
cd backend
npm install @vitalets/google-translate-api franc-min
```

## Step 2: Restart Backend Server

```bash
npm run dev
```

## Step 3: Test the Feature

1. Navigate to any donation detail page
2. Look for the translate button (🌐 icon) next to "Description"
3. Click it to translate the description
4. Click the undo button (↶ icon) to show original text

## That's it! ✅

The translation feature is now ready to use.

For more details, see `TRANSLATION_SETUP_INSTRUCTIONS.md`
